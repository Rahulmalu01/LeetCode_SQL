# LeetCode 181 - Employees Earning More Than Their Managers
## Intuition
Each employee record contains a managerId that points to another employee in the same table. To determine whether an employee earns more than their manager, we need to compare the employee's salary with the salary of the employee whose id matches the employee's managerId.

A self-join allows us to access both the employee's and manager's information in a single query, making the solution more efficient and easier to understand.

## Approach
1. Perform a self-join on the Employee table:
    - e1 represents the employee.
    - e2 represents the manager.

2. Match employees with their managers using:
```
e1.managerId = e2.id
```
3. Filter employees whose salary is greater than their manager's salary.
4. Return the employee's name as Employee.

## Complexity
- Time complexity: O(n)

- Space complexity: O(1)

## Code
```mysql []
SELECT e1.name AS Employee
FROM Employee e1
JOIN Employee e2
    ON e1.managerId = e2.id
WHERE e1.salary > e2.salary;
```

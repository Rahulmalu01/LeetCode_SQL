# LeetCode 620 - Not Boring Movies

For LeetCode **620. Not Boring Movies**, you can use the following post:

## # Intuition

The problem asks us to find movies that:

1. Have an **odd-numbered ID**.
2. Have a description that is **not "boring"**.
3. Are sorted by **rating in descending order**.

Since all the required information is already present in the `Cinema` table, we only need to filter the rows based on the given conditions and then sort the results.

## # Approach

* Use the modulo operator (`%`) to identify odd IDs:

  ```sql
  id % 2 = 1
  ```
* Exclude movies whose description is `"boring"`.
* Sort the remaining movies by `rating` in descending order using `ORDER BY`.

## # Complexity

* **Time Complexity:** `O(n log n)` due to sorting by rating.
* **Space Complexity:** `O(1)` (excluding the output result).

## # Code

```mysql
# Write your MySQL query statement below
SELECT *
FROM Cinema
WHERE id % 2 = 1
  AND description <> 'boring'
ORDER BY rating DESC;
```

### Why this works

* `id % 2 = 1` keeps only odd-numbered movie IDs.
* `description <> 'boring'` removes all boring movies.
* `ORDER BY rating DESC` ensures the highest-rated movies appear first.

# LeetCode 584 - Find Customer Referee

# Intuition

We need to find the names of customers who were not referred by the customer with id = 2.

A customer can have:

A referee whose ID is stored in referee_id.
No referee at all (NULL).

Since customers with NULL referee IDs are also not referred by customer 2, they should be included in the result.

# Approach
Select the name column from the Customer table.
Filter out customers whose referee_id is 2.
Include customers whose referee_id is NULL because they do not have a referee.

The condition:
```
referee_id != 2 OR referee_id IS NULL
```
ensures that both requirements are satisfied.

# Complexity
- Time Complexity: O(n) — we scan each customer record once. 
- Space Complexity: O(1) — no extra space is used apart from the output.
# Code
```
SELECT name
FROM Customer
WHERE referee_id != 2
   OR referee_id IS NULL;
```

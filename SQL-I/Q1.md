# LeetCode 175 - Combine Two Tables
## Intuition
We need to display information from both tables:

- Person contains firstName and lastName.
- Address contains city and state.

The key relationship between the tables is personId.

Since the problem requires all people to appear in the result, even if they don't have an address, we cannot use an INNER JOIN because it would exclude people without matching records in the Address table.

A LEFT JOIN is the appropriate choice because:

1. It keeps every row from the Person table.
2. It adds the corresponding city and state from the Address table when a match exists.
3. If no address is found, the address columns are returned as NULL.

## Approach
1. Start with the Person table since every person must be included
2. Perform a LEFT JOIN with the Address table using personId.
3. Select the required columns: firstName, lastName, city, and state.

## Complexity
- Time complexity: O(n + m)

- Space complexity: O(1)

## Code
```mysql []
SELECT p.firstName, p.lastName, a.city, a.state From Person p
LEFT OUTER JOIN Address a ON p.personId = a.personId;
```
